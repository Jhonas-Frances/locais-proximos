# 📍 Locais Próximos

Aplicativo mobile desenvolvido em React Native que permite encontrar estabelecimentos próximos utilizando a API do Google Maps.

## 🚀 Funcionalidades

- Buscar estabelecimentos próximos à localização do usuário.
- Filtrar por categoria (mercados, farmácias, restaurantes, postos de combustível, entre outros).
- Definir o raio de busca (ex.: 1 km, 5 km, 10 km).
- Exibir os resultados com base na localização atual do usuário.

## 🛠️ Tecnologias utilizadas

- React Native
- JavaScript
- Google Maps API
- Geolocalização (GPS)

## 🎯 Objetivo

Este projeto foi desenvolvido com o objetivo de praticar o consumo de APIs, geolocalização em dispositivos móveis e desenvolvimento de aplicações utilizando React Native.

## 📱 Como funciona

1. O usuário permite o acesso à localização.
2. Seleciona uma categoria de estabelecimento.
3. Define o raio de busca.
4. O aplicativo consulta a API do Google Maps e exibe os locais encontrados nas proximidades.
